# IA — REMOVIDA DO ESCOPO

> **Decisão:** Todas as features de IA foram removidas do projeto IRP a pedido do cliente.

## Features removidas

| Feature original | Motivo da remoção |
|------------------|-------------------|
| Validação IA de docs aduaneiros (Fase 5) | Complexidade alta, risco fiscal, custo desnecessário |
| Agente IA de CRM (M2) | Escopo vago, baixo valor imediato |

## Impacto

- **MVP simplificado**: 5 módulos CRUD, zero dependência de LLM
- **Custo reduzido**: sem gasto com API de IA (~R$ 0/mês vs ~R$ 10/mês estimado)
- **Prazo menor**: ~5-6 semanas MVP vs ~7-8 com IA
- **Risco menor**: sem chance de alucinação em valores fiscais

---

Caso o cliente queira reavaliar IA no futuro, a estratégia completa está documentada nos commits anteriores deste arquivo.
